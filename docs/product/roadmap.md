## 一、MVP 定义

### 1.1 MVP 范围（P0）

**目标：** 4周内交付可用的最小产品

| 模块 | 功能 | 说明 |
|------|------|------|
| 渠道管理 | 新增/编辑/删除/测试 | 仅支持 OpenAI 格式渠道 |
| API 密钥管理 | 生成/查看/删除 | 支持配额限制 |
| API 代理 | 请求转发 | 仅 /chat/completions |
| 基础统计 | 请求数/Token数 | 按日统计 |
| 用户系统 | 注册/登录 | 仅邮箱登录 |

**MVP 不包含：**
- 多渠道负载均衡
- 高级计费
- 移动端适配
- 第三方登录
