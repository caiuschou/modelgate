# ModelGate 用户控制台

**版本:** 1.0
**更新日期:** 2026年4月

---

## 一、产品概述

ModelGate 用户控制台是面向管理员、开发者和运维人员的管理门户。它提供统一账号管理、API 令牌控制、用量监控、渠道配置和审计日志功能，帮助团队在一个安全可控的界面中管理大模型接入和使用。

**核心价值：**
- 统一入口：集中管理令牌、用户、渠道和用量
- 可视化：实时展示使用统计、费用和异常告警
- 安全可控：权限分级、IP/域名限制、审计追踪
- 灵活扩展：支持多模型供应商、多租户和多团队场景

---

## 二、目标用户

| 用户角色 | 主要职责 | 核心需求 |
|---------|---------|---------|
| 系统管理员 | 账号、令牌、渠道管理 | 权限配置、审计、安全、成本控制 |
| 开发者 | API 访问与令牌使用 | 快速获取令牌、查看用量、调试错误 |
| 运营人员 | 费用分析与使用监控 | 用量统计、预算告警、报表导出 |
| 安全审计人员 | 请求审计与合规检查 | 日志检索、异常追踪、导出报告 |

## 三、核心功能

### 3.1 仪表盘

**主要内容：**
- 总体用量概览：请求数、Token 消耗、成本、成功率
- 今日/本周/本月趋势图
- 令牌使用排行、模型调用排行
- 异常事件：错误率、超时、渠道故障
- 预算告警与配额预警

**功能目的：**
- 一屏掌握服务运行态势
- 及时发现调用异常和成本爆发
- 快速定位高频使用令牌与模型

### 3.2 账号与权限管理

**功能点：**
- 用户账号管理：新增、编辑、禁用、删除
- 角色配置：管理员、普通用户、审计员
- 多租户与团队管理：组织结构、成员邀请
- 登录方式：用户名/邮箱、第三方登录（扩展）

**权限控制：**
- 管理接口权限与查询权限分离
- 只允许特定角色访问敏感运营功能
- 支持只读账号查看日志与报表

### 3.3 令牌管理

**功能点：**
- 令牌创建：命名、描述、访问权限、过期时间
- 令牌分组与标签管理
- 配额设置：Token 数量、金额、时段限制
- 权限规则：可访问模型范围、API 类型限制
- 令牌状态：启用、禁用、过期、撤销

**安全机制：**
- 令牌仅生成一次，后端加密存储
- 支持 IP 白名单与域名限制
- 令牌操作日志记录

### 3.4 渠道与模型管理

**功能点：**
- 渠道配置：供应商、API Key、模型映射、权重
- 渠道分组与标签：按业务或部门分类
- 健康检查：通断检测、响应时长、错误率
- 调度策略：优先级、加权、最少连接
- 灰度发布：按用户组或比例逐步开放渠道

**目标效果：**
- 保障调用高可用
- 快速切换故障渠道
- 支持多渠道并行与负载均衡

### 3.5 用量统计与分析

**展示指标：**
- 请求数、输入 Token、输出 Token、总 Token
- 成本（金额）、平均延迟、成功率
- 模型/令牌/用户/渠道维度分布
- 时序趋势、同比/环比分析

**分析能力：**
- 自定义时间范围筛选
- 支持按模型、令牌、用户、渠道分组
- 支持导出 CSV/JSON 报表
- 支持预算阈值告警

### 3.6 审计日志与异常排查

**功能点：**
- 全链路请求日志记录（请求、响应、用量、成本）
- 按时间、用户、令牌、渠道、模型、状态码检索
- 异常事件过滤：错误、超时、失败率突增
- 日志导出：JSON、CSV
- 操作审计：谁在什么时候进行了哪些管理操作

**合规特性：**
- 日志不可篡改或只读配置
- 日志保留策略可配置
- 支持企业级合规审计需求

## 四、用户场景

### 4.1 新用户快速上手

1. 管理员登录控制台
2. 创建新 API 令牌并设置访问范围
3. 将令牌分发给开发者
4. 监控首日用量与错误情况

### 4.2 管理员分发与管控令牌

1. 创建令牌并绑定模型访问权限
2. 设置 IP 白名单与域名限制
3. 按需调整配额与过期时间
4. 通过仪表盘查看令牌使用情况

### 4.3 运营监控成本与预算

1. 在仪表盘查看全局费用趋势
2. 发现成本异常后，定位高消费令牌/模型
3. 调整配额或禁用高风险令牌
4. 导出当月成本报表

### 4.4 审计与问题排查

1. 通过审计日志检索异常请求
2. 确认请求来源用户与令牌
3. 查看上游模型、错误信息、耗时数据
4. 记录处理结果并归档日志

## 五、设计原则

- 简洁直观：以仪表盘与卡片形式展示核心指标
- 可控可靠：权限、审计、告警三层防护
- 统一标准：令牌与渠道统一管理，避免碎片化
- 业务友好：支持多租户、多团队、多级组织结构
