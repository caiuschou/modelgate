# ModelGate 导航设计

**版本:** 1.1  
**日期:** 2026年4月

---

## 当前控制台实现（2026-04）

与 `frontend/src/components/layout/app-layout.tsx` 及 `frontend/src/routes/index.tsx` 一致：

| 侧边栏文案 | 路径 | 实现 |
|------------|------|------|
| 首页 | `/` | 仪表盘页 |
| 渠道管理 | `/channels` | 占位（管理员） |
| 令牌管理 | `/tokens` | 占位 |
| 用户管理 | `/users` | 占位（管理员） |
| 日志中心 | `/logs` | 列表与详情 |
| 统计分析 | `/analytics` | 占位 |
| 系统设置 | `/settings` | 占位（管理员） |

顶栏含主题切换、用户名、退出；**无**规格书中的全局搜索框、通知中心、用户头像下拉（可后续迭代）。

## 一、全局导航

### 1.1 导航结构

**顶部栏（固定）：**
```
[Logo] ModelGate          [搜索框]          [通知] [用户头像 ▼]
```

**侧边栏（可折叠）：**
```
┌─────────────────┐
│ 🏠 首页          │
│ 🔌 渠道管理      │
│ 🔑 令牌管理      │
│ 👥 用户管理      │
│ 📊 统计分析      │
│ 📝 日志中心      │
│ 💰 充值中心      │
│ ⚙️ 系统设置      │
└─────────────────┘
```

### 1.2 导航交互

| 交互 | 行为 |
|------|------|
| 点击侧边栏项 | 展开子菜单（如有）/ 跳转页面 |
| Hover 侧边栏项 | 背景变浅，显示 Tooltip（折叠时） |
| 点击折叠按钮 | 侧边栏收起为图标模式 |
| 点击用户头像 | 下拉菜单：个人设置 / 退出登录 |
| 点击通知图标 | 展开通知面板 |

### 1.3 侧边栏折叠状态

**展开状态：**
- 宽度：200px
- 显示：图标 + 文字
- 过渡：300ms ease-out

**折叠状态：**
- 宽度：64px
- 显示：仅图标
- Hover 显示 Tooltip

## 二、页面标题栏

### 2.1 标准标题栏

```
┌──────────────────────────────────────────────────────────────┐
│  首页 > 概览                          [刷新]                 │
├──────────────────────────────────────────────────────────────┤
│  面包屑导航           操作按钮                              │
└──────────────────────────────────────────────────────────────┘
```

### 2.2 面包屑规范

- **格式：** 首页 > 一级 > 二级 > 当前
- **点击：** 除当前页外均可点击跳转
- **分隔符：** `>`
- **样式：** 灰色文字，可点击项蓝色

### 2.3 操作按钮

**位置：** 右上角
**间距：** 按钮之间 8px
**主要操作：** 使用主按钮样式
**次要操作：** 使用次按钮样式

## 七、页面跳转

### 7.1 页面切换动画

**类型：** 淡入淡出

```
离开页面：opacity 1 → 0, 200ms ease-out
进入页面：opacity 0 → 1, 200ms ease-out
```

**嵌套路由切换：**
```
离开：translateX(0) → translateX(-20px) + opacity 1 → 0
进入：translateX(20px) → translateX(0) + opacity 0 → 1
时长：250ms
```

### 7.2 返回行为

- 浏览器返回按钮：正常返回
- 面包屑点击：跳转到指定层级
- 左侧导航点击：切换到对应页面
