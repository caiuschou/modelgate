# ModelGate 移动端设计

**版本:** 1.2  
**日期:** 2026年4月

> **实现说明：** 当前控制台以**桌面端**为主；移动端专项交互尚未完全落地，本文为规划参考。

## 一、移动端导航

### 1.1 底部导航栏

**布局：**
```
┌────────────────────────────────────┐
│                                    │
│           页面内容                  │
│                                    │
├────────────────────────────────────┤
│  🏠      📊      🔑      👤       │
│  首页    统计  API 密钥  我的       │
└────────────────────────────────────┘
```

**规范：**
- 高度：56px
- 图标：24px
- 文字：12px
- 选中：蓝色高亮 `#1890FF`

**导航项：**
- 🏠 首页
- 📊 统计
- 🔑 API 密钥
- 👤 我的

### 1.2 侧边抽屉

**触发：** 点击顶部汉堡菜单

```
┌──────────────────────────┐

│  💰 充值中心              │
```

**规范：**
- 宽度：280px

## 三、表单移动端优化

### 3.1 输入框优化

- 高度增大：40px（更易点击）
- 字号增大：16px（防止 iOS 自动缩放）
- 标签置顶（非左侧）

**示例：**
```
┌────────────────────────────────────┐
│ 渠道名称                           │
│ ┌────────────────────────────────┐ │
│ │                                │ │
│ └────────────────────────────────┘ │
│                                    │
│ API 密钥                           │
│ ┌────────────────────────────────┐ │
│ │                                │ │
│ └────────────────────────────────┘ │
└────────────────────────────────────┘
```

### 3.2 选择器优化

- 使用原生选择器（iOS/Android 风格）
- 日期使用原生日期选择器

### 3.3 按钮优化

- 全宽按钮（更易点击）
- 底部固定按钮区域

```
┌────────────────────────────────────┐
│                                    │
│           表单内容                  │
│                                    │
│                                    │
├────────────────────────────────────┤
│     [取消]          [保存]         │
└────────────────────────────────────┘
```

## 四、触控交互规范

### 4.1 最小点击区域

**规范：** 44×44px

### 4.2 手势支持

| 手势 | 操作 |
|------|------|
| 下拉 | 刷新列表 |
| 左滑 | 显示删除按钮（列表项） |
| 长按 | 显示上下文菜单 |
| 双击 | 无 |

### 4.3 触摸反馈

- 按下：背景变深 10%
- 释放：恢复原状
- 取消：恢复原状
