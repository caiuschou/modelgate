**版本:** 1.0  
**日期:** 2026年3月  
**设计师:** AI Design Team  
**文档类型:** 交互设计规格说明书

---

## 一、设计概述

### 1.1 设计目标

- **高效操作** — 核心任务 3 步内完成
- **信息清晰** — 关键数据一目了然
- **降低门槛** — 新用户 5 分钟上手
- **专业可信** — 企业级视觉品质

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 一致性 | 相同功能采用一致的交互模式 |
| 反馈即时 | 操作后 200ms 内给予反馈 |
| 容错性 | 危险操作需二次确认，支持撤销 |
| 渐进披露 | 高级功能折叠，不干扰主流程 |

### 1.3 用户角色

| 角色 | 权限 | 核心任务 |
|------|------|---------|
| 超级管理员 | 全部 | 渠道配置、用户管理、系统设置 |
| 普通管理员 | 部分 | 令牌管理、查看统计 |
| 普通用户 | 基础 | 查看配额、使用 API |

### 3.2 导航交互

| 交互 | 行为 |
|------|------|
| 点击侧边栏项 | 展开子菜单（如有）/ 跳转页面 |

## 五、通用组件规范

### 5.1 按钮规范

| 类型 | 样式 | 使用场景 |
|------|------|---------|
| 主按钮 | 蓝色背景 + 白字 | 主要操作（保存、提交） |
| 次按钮 | 白色背景 + 蓝边框 | 次要操作（取消、重置） |
| 文字按钮 | 无背景 + 蓝字 | 低优先级操作 |
| 危险按钮 | 红色背景 + 白字 | 删除、禁用等危险操作 |
| 图标按钮 | 仅图标 | 表格操作、工具栏 |

**状态：**
- Hover：背景色加深 10%
- Disabled：透明度 50%，禁用点击
- Loading：显示旋转图标，禁用点击

### 5.2 表单组件规范

| 组件 | 规范 |
|------|------|
| 输入框 | 高度 36px，圆角 4px，边框 1px |
| 下拉框 | 与输入框一致，右侧下拉箭头 |
| 复选框 | 16×16px，选中显示勾选图标 |
| 单选框 | 16×16px 圆形，选中显示实心圆 |
| 开关 | 宽 44px，高 22px |
| 日期选择 | 弹出日历面板 |

**校验状态：**
- 默认：灰色边框
- 聚焦：蓝色边框 + 阴影
- 错误：红色边框 + 错误提示文字
- 成功：绿色边框（可选）

### 5.3 表格规范

| 元素 | 规范 |
|------|------|
| 表头 | 背景色 #F5F5F5，字重 500 |
| 表格行 | 高度 48px，斑马纹（可选） |
| Hover | 背景色 #FAFAFA |
| 选中 | 背景色 #E6F7FF |
| 操作列 | 固定右侧，图标按钮 |

### 5.4 弹窗规范

| 类型 | 尺寸 | 使用场景 |
|------|------|---------|
| 小弹窗 | 400px | 确认对话框 |
| 中弹窗 | 600px | 表单弹窗 |
| 大弹窗 | 800px | 复杂表单/详情 |
| 抽屉 | 右侧滑出 400-600px | 详情展示 |

**结构：**
- 标题栏：20px 字号，右侧关闭按钮
- 内容区：最大高度 70vh，超出滚动
- 底部栏：右对齐操作按钮

### 5.5 Toast 提示规范

| 类型 | 颜色 | 使用场景 |
|------|------|---------|
| 成功 | 绿色 | 操作成功 |
| 错误 | 红色 | 操作失败 |
| 警告 | 橙色 | 需要注意 |
| 信息 | 蓝色 | 一般提示 |

**规范：**
- 位置：页面顶部居中
- 自动消失：3 秒
- 可手动关闭
