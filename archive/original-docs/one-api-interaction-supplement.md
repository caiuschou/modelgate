## 三、异常流程设计

### 3.1 网络异常

**全局网络断开**

```
┌────────────────────────────────────────────────────────────┐
│ ⚠️ 网络连接已断开，正在尝试重连...              [刷新页面] │
└────────────────────────────────────────────────────────────┘
```

- 位置：页面顶部固定条
- 背景：`#FFFBE6`
- 自动重连：每 5 秒尝试一次
- 重连成功：显示 Toast "网络已恢复"

**请求失败**

```
┌────────────────────────────────────┐
│           ❌                       │
│     数据加载失败                    │
│   请检查网络连接后重试               │
│                                    │
│        [ 重新加载 ]                 │
└────────────────────────────────────┘
```

### 3.2 会话过期

**弹窗提示**

```
┌────────────────────────────────────┐
│  会话已过期                   [×]  │
├────────────────────────────────────┤
│                                    │
│    🔐 您的登录已过期                │
│    请重新登录以继续使用             │
│                                    │
│                  [取消]  [重新登录] │
└────────────────────────────────────┘
```

- 触发：API 返回 401
- 行为：点击重新登录跳转登录页
- 记忆：保存当前页面 URL，登录后跳回

## 六、动画详细说明

### 6.1 页面切换

**类型：** 淡入淡出

```
离开页面：opacity 1 → 0, 200ms ease-out
进入页面：opacity 0 → 1, 200ms ease-out
```

**嵌套路由切换**

```
离开：translateX(0) → translateX(-20px) + opacity 1 → 0
进入：translateX(20px) → translateX(0) + opacity 0 → 1
时长：250ms
```

### 6.2 弹窗动画

**入场**

```
遮罩：opacity 0 → 1, 200ms ease-out
弹窗：scale(0.9) opacity 0 → scale(1) opacity 1, 250ms ease-out
```

**出场**

```
遮罩：opacity 1 → 0, 200ms ease-in
弹窗：scale(1) opacity 1 → scale(0.9) opacity 0, 200ms ease-in
```

### 6.3 抽屉动画

**入场**

```
遮罩：opacity 0 → 1, 200ms ease-out
抽屉：translateX(100%) → translateX(0), 300ms ease-out
```

**出场**

```
遮罩：opacity 1 → 0, 200ms ease-in
抽屉：translateX(0) → translateX(100%), 300ms ease-in
```

### 6.4 Toast 动画

**入场**

```
transform: translateY(-100%) → translateY(0)
opacity: 0 → 1
时长：250ms ease-out
```

**出场**

```
transform: translateY(0) → translateY(-100%)
opacity: 1 → 0
时长：250ms ease-in
```

### 6.5 列表加载

**骨架屏**

```
┌────────────────────────────────────┐
│ ┌────────────────────────────────┐ │
│ │ ████████████  ████             │ │
│ │ ████████  ████████████         │ │
│ │ ████  ████████████  ████       │ │
│ └────────────────────────────────┘ │
└────────────────────────────────────┘
```

- 骨架颜色：`#F0F0F0`
- 闪烁动画：opacity 0.5 → 1 → 0.5, 1.5s infinite

**渐进加载**

```
新数据进入：opacity 0 → 1, translateY(10px) → translateY(0), 300ms
```

### 6.6 删除确认动画

**行删除**

```
1. 点击删除：行抖动 2 次（translateX ±4px, 100ms × 4）
2. 确认删除：高度收起 height → 0, 300ms
3. 删除完成：淡出 opacity → 0, 200ms
```

### 6.7 下拉刷新

```
┌────────────────────────────────────┐
│           ↓ 下拉刷新               │

- 图标旋转：360deg, 1s linear infinite

```
